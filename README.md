# Goomti Restaurant — React Dashboard

Live interactive sales analytics dashboard built with React, 
using real restaurant POS and delivery platform data.

## 🔗 Live Demo
[View Dashboard](https://goomti-react-dashboard-ihxd.vercel.app)

## Pages
- **Overview** — Daily sales, KPIs, day-of-week patterns
- **Channels** — Revenue breakdown by platform (POS, Uber Eats, Deliveroo etc.)
- **Menu** — Top 10 items by revenue, category performance
- **Customers** — Retention analysis, opportunity model, industry benchmark

## Tech Stack
React · Recharts · PapaParse · JavaScript · CSS

## Data
Real sales data from Goomti Restaurant (Inaya Hospitality Group)
covering May–June 2026 across 6 channels and 112 menu items.
